# CubeMX 配置

一共需要配置13个引脚，任意空闲普通io都可以

```c
extern driverIO SDIO0;
extern driverIO SDIO1;
extern driverIO SDIO2;
extern driverIO SDIO3;
extern driverIO PDC;
extern driverIO RST;
extern driverIO SCLK;
extern driverIO CS;
extern driverIO UPDATE;
extern driverIO PS0;
extern driverIO PS1;
extern driverIO PS2;
extern driverIO PS3;
```

需要在CubeMX里面将GPIO设置为GPIO_OUTPUT模式，然后自定义IO名称为上述的**SDIO0**之类的

---

>一定要5V供电，不能用正点原子的12VDC供电！！！！！！！！！！！

AD9959驱动使用示例   

## 外部接口

| 函数 | 说明 |
|------|------|
| `void Init_AD9959(void)` | 初始化AD9959（PLL配置、四通道默认参数） |
| `void Write_Frequence(uint8_t Channel, uint32_t Freq)` | 设置通道频率，`Channel` 取值 0~3，`Freq` 单位 Hz（0~50MHz） |
| `void Write_Amplitude(uint8_t Channel, uint16_t Ampli)` | 设置通道幅值，`Ampli` 取值 0~1023 |
| `void Write_Phase(uint8_t Channel, uint16_t Phase)` | 设置通道相位，`Phase` 取值 0~16383 对应 0~360° |
| `void AD9959_IO_Update(void)` | 发送 IO UPDATE 脉冲，设置完参数后必须调用才生效 |

## 使用示例

初始化AD9959，设置通道2输出正弦信号（增益1023、相位0、频率50kHz）：

```c
Init_AD9959();
Write_Frequence(2, 50000);
Write_Amplitude(2, 1023);
Write_Phase(2, 0);
AD9959_IO_Update();  // 设置完参数必须调用此函数才会生效
```

## CMake 工程配置

```cmake
# 添加驱动源文件
target_sources(${CMAKE_PROJECT_NAME} PRIVATE
    MY_DRIVE/MY_DDS/ad9959.c
)

# 添加驱动头文件路径
target_include_directories(${CMAKE_PROJECT_NAME} PRIVATE
    MY_DRIVE/MY_DDS
)
```
