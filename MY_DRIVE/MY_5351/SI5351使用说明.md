# 使用说明

## Hal库配置IIC

`MX_I2C1_Init()` — CubeMX 中使能 I2C1，驱动内部固定使用 `hi2c1`

## 导入4个文件

`si5351.c` / `si5351.h` / `si5351_errors.h` / `si5351_asserts.h`

main函数添加

```c
#include "si5351.h"
// 以下宏由用户在 main.c 中自行定义（驱动头文件中不包含）
#define MHZ(x)    ((uint32_t)((x) * 1000000UL))
#define KHZ(x)    ((uint32_t)((x) * 1000UL))
#define HZ(x)     ((uint32_t)(x))
// 初始化
si5351_Init();
```

## 外部接口

| 函数 | 说明 |
|------|------|
| `err_t si5351_Init(void)` | 初始化 Si5351 芯片和内部状态，使用前必须调用 |
| `err_t SI5351_SetFrequency(uint8_t clk, uint32_t fout)` | 设置通道频率，`clk` 取值 0/1/2，`fout` 单位 Hz |
| `err_t SI5351_SetChannelPower(uint8_t channel, uint8_t enabled)` | 开关通道输出，`enabled` = 0 关 / 1 开 |

## 使用示例

```c
si5351_Init();
SI5351_SetFrequency(0, MHZ(120));   // CLK0 → 120MHz
SI5351_SetFrequency(1, KHZ(500));   // CLK1 → 500kHz
SI5351_SetChannelPower(2, 0);       // 关闭 CLK2
```

## CMake 工程配置

```cmake
# 添加驱动源文件
target_sources(${CMAKE_PROJECT_NAME} PRIVATE
    MY_DRIVE/MY_5351/si5351.c
)

# 添加驱动头文件路径
target_include_directories(${CMAKE_PROJECT_NAME} PRIVATE
    MY_DRIVE/MY_5351
)
```
